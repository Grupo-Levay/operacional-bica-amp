# Coding Standards

- Arquivo único index.html — sem dependências externas, sem build
- CSS com variáveis (--bica, --amp, --ok, --warn, --danger)
- JS vanilla com localStorage para estado
- Funções prefixadas por domínio: renderEstoque(), syncCompras(), etc.
- Mobile-first: touch targets mínimos de 52px
- Commits: conventional commits (feat:, fix:, chore:)
