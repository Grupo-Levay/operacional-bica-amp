# XOIA Memory

Sistema de memória leve — arquivos locais, zero dependências.

## Estrutura

```
memory/
  learnings.md      — aprendizados persistentes do projeto
  sessions.jsonl    — log de ciclos executados
  clients/
    bica.md         — padrões e preferências do cliente Bica/AMP
```

## Como usar

Leia `learnings.md` e o arquivo do cliente antes de iniciar um ciclo.
Escreva ao final de `/investigate` (fix) ou `/design-shotgun` (escolha).
Veja `.claude/rules/memory.md` para regras completas.
